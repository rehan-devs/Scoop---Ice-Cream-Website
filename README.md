## ✨ Features

- 🎨 Luxury design with warm, creamy color palette
- 🍨 Interactive 3D ice cream cone (React Three Fiber)
- 🌀 Buttery-smooth scroll animations (GSAP + Lenis)
- ♾️ Infinite carousel with auto-scroll
- 📱 Fully responsive
- ⚡ Optimized performance
- ♿ Accessible

## 🛠️ Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Animations:** GSAP + ScrollTrigger
- **3D:** React Three Fiber + Drei
- **Smooth Scroll:** Lenis

## 🚀 Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
